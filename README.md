# Ke Shi academic website starter

This is a plain static website for GitHub Pages. It has no build system, no package manager, and no framework.

## Files

- `index.html` — the content of the website
- `style.css` — the visual design
- `assets/` — place PDFs, photos, and other files here later

## How to publish on GitHub Pages

1. Create or sign in to a GitHub account.
2. Create a new public repository named exactly:
   `<your-github-username>.github.io`
3. Upload `index.html`, `style.css`, and the `assets` folder into the repository.
4. Go to `Settings` → `Pages`.
5. Under `Build and deployment`, choose `Deploy from a branch`.
6. Set the branch to `main` and the folder to `/root`.
7. Save. Your site should publish at:
   `https://<your-github-username>.github.io`

## Later edits

- Replace the email address in `index.html` if needed.
- Upload a CV as `assets/Ke_Shi_CV.pdf` and change the CV button to link to it.
- Add a profile photo by placing it in `assets/` and replacing the `portrait-placeholder` block with an `<img>` tag.
